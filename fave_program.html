Leaves of Grass Program

This Python program scrapes a website for the classic Walt Whitman poetry book, "Leaves of Grass", and downloads it to a text file for offline reading.

Snapshot of the Leaves of Grass program code:


                import requests
                whitman1 = requests.get('http://www.gutenberg.org/files/1322/1322-0.txt')
                whitman1.raise_for_status()
                print(whitman1.text[500:750])
                grassfile = open('leaves_of_grass.txt','wb')
                for chunk in whitman1.iter_content(100000):
                    grassfile.write(chunk)
                grassfile.close()
